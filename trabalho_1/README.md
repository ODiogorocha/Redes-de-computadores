# Objetivos

Entender a estrutura do basic.p4 (https://github.com/p4lang/tutorials/tree/master/exercises/basic)

Executar o exemplo basic no BMv2/Mininet.
Modificar a topologia para 1 switch conectando 2 hosts (simplificar a topologia).

Alterar o código P4 para encaminhar e decrementar o TTL de pacotes ICMP. Pacotes de outros protocolos devem ser bloqueados.
Criar/ajustar regras de tabela de acordo com as eventuais mudanças no código.

# Entregáveis

* topology.json modificado (renomeado para topo_trabalho1.json)

* Código P4 alterado (renomeado para basic_trabalho1.p4)

* Relatório curto (1-2 páginas) indicando o que foi modificado, como executar (comandos) e prints dos resultados

* Não esquecer de informar o nome completo de todos os integrantes do grupo em um arquivo "grupo.txt"
